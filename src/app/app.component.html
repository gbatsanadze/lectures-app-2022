სემინარის დავალება

  1. გახადე shell და auth მოდულები lazy loaded.
  2. გამოიყენე საკუთარი preloading სტრატეგიისთვის კლასი, რომელიც შეხედვს route.data.preload ველის მნიშვნელობას რათა განსაზღვროს ჩატვირთოს თუ არა ესა თუ ის როუტი წინასწარ.
  3. გახადე auth მოდული preloaded lazy მოდული.
  4. გაიტანე API სერვერის ლინკი (რომელიც ახლა ინტერსეპტორში წერია) environment ცვლადებში.
  5. დაბილდე აპლიკაცია --prod flag-ით.
